# 移动端使用指南

## 📱 概述

本应用已完全优化移动端体验，支持在任何移动设备上访问和使用：

- ✅ iOS / Android 手机
- ✅ 平板电脑
- ✅ 响应式设计
- ✅ 触摸优化
- ✅ 深色主题

## 🚀 快速开始

### 1. 访问应用

在手机浏览器输入应用地址：
```
https://your-app.vercel.app
```

### 2. 添加到主屏幕

#### iOS (iPhone/iPad)
1. 在 Safari 中打开应用
2. 点击底部的分享按钮 (📤)
3. 选择"添加到主屏幕"
4. 点击"添加"

现在可以从主屏幕直接启动，像原生应用一样使用！

#### Android (Chrome)
1. 在 Chrome 中打开应用
2. 点击右上角菜单 (⋮)
3. 选择"添加到主屏幕"
4. 点击"添加"

## 🎯 核心功能

### 策略回测

1. 点击底部"回测"标签
2. 选择交易对（如 BTCUSDT）
3. 配置策略参数
4. 点击"开始回测"
5. 查看回测结果图表

### 自动交易

1. 点击底部"交易"标签
2. 配置币安 API 密钥
3. 选择交易策略
4. 设置交易参数
5. 启动自动扫描

## 💡 使用技巧

### 1. 横屏模式

对于图表查看，横屏模式体验更好：
- iOS：控制中心 → 屏幕旋转锁定
- Android：设置 → 显示 → 自动旋转

### 2. 浏览器选择

推荐使用以下浏览器：
- **iOS**: Safari（最佳体验）
- **Android**: Chrome 或 Edge
- **其他**: 任何现代浏览器

### 3. 网络要求

- ✅ 稳定的网络连接
- ✅ 建议使用 Wi-Fi 或 4G/5G
- ⚠️ WebSocket 连接需要持续网络

### 4. 电池优化

为了保持 WebSocket 连接稳定：
- iOS：设置 → Safari → 关闭"阻止跨站跟踪"
- Android：Chrome 设置 → 省电模式 → 排除此应用

## ⚙️ 触摸操作

### 导航
- **滑动切换标签**：在移动端点击底部导航
- **滚动内容**：上下滑动浏览内容区域
- **按钮点击**：所有按钮都已优化触摸区域

### 图表操作
- **缩放**：双指捏合
- **平移**：单指拖动
- **十字线**：点击图表查看详细数据

### 输入操作
- **数字输入**：自动弹出数字键盘
- **日期选择**：使用系统日期选择器
- **下拉选择**：使用系统选择器

## 🔒 安全提示

### API 密钥存储

- ✅ API 密钥存储在您的设备本地
- ✅ 不会上传到应用服务器
- ✅ 清除浏览器数据会删除密钥

### 浏览器安全

1. **使用 HTTPS**：确保通过 HTTPS 访问
2. **不要截图分享**：避免在截图中泄露 API 密钥
3. **定期更新**：保持浏览器和系统最新

### 设备安全

1. **使用锁屏密码**
2. **不要在公共设备上保存密钥**
3. **退出后清除浏览数据**

## 📊 性能优化

### 减少数据加载

- 使用稳定的网络连接
- 避免频繁切换页面
- 适当增加扫描间隔

### 内存管理

- 关闭不需要的浏览器标签
- 定期清理浏览器缓存
- 重启应用释放内存

### WebSocket 连接

- 应用后台运行时连接可能断开
- 返回应用时会自动重连
- 网络切换时自动重连

## 🐛 常见问题

### Q: 底部导航栏不见了？

**A**: 可能是因为：
1. 页面内容加载中，等待几秒
2. 刷新页面：下拉刷新
3. 清除浏览器缓存

### Q: 图表无法显示？

**A**: 检查：
1. 网络连接是否正常
2. 尝试横屏查看
3. 刷新页面

### Q: WebSocket 连接断开？

**A**:
1. 检查网络连接
2. 返回应用，会自动重连
3. 切换到更好的网络（如 Wi-Fi）

### Q: 交易指令没有执行？

**A**:
1. 检查 API 密钥是否有效
2. 确认账户余额充足
3. 检查网络连接
4. 查看系统日志

### Q: 画面显示不正常？

**A**:
1. 清除浏览器缓存
2. 更新浏览器到最新版本
3. 尝试其他浏览器
