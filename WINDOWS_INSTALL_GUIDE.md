# Windows 桌面应用安装指南

本项目提供两个自动化脚本，帮助你在 Windows 系统上快速构建和安装币安自动交易桌面应用。

## 前置要求

1. **Windows 10/11 (x64)**
2. **Node.js** (v18 或 v20) - [下载地址](https://nodejs.org)
3. **Git** - [下载地址](https://git-scm.com/download/win)
4. **pnpm** - 安装 Node.js 后运行：`npm install -g pnpm`

## 快速安装方式

### 方式 1：全新安装（推荐）

如果你还没有项目代码，或想从头开始：

1. **下载 `fresh-install.bat` 脚本**
   - 从 GitHub 下载 `fresh-install.bat` 文件
   - 将文件放到任意位置（如桌面）
   - 双击运行

2. **脚本会自动执行以下步骤：**
   - 删除旧的项目目录
   - 从 GitHub 克隆最新代码
   - 安装所有依赖
   - 构建 Windows 桌面应用

3. **等待完成（约 10-15 分钟）**
   - 构建成功后，安装包位于：`C:\Users\Administrator\Desktop\bushu1\dist\币安自动交易 Setup 0.1.0.exe`

### 方式 2：重新构建

如果你已经有项目代码，只想重新构建：

1. **下载 `rebuild.bat` 脚本**
   - 从 GitHub 下载 `rebuild.bat` 文件
   - 将文件放到项目根目录（`bushu1` 文件夹内）
   - 双击运行

2. **脚本会自动执行以下步骤：**
   - 清理缓存和旧的构建产物
   - 重新安装依赖
   - 构建桌面应用

3. **等待完成（约 5-10 分钟）**

## 安装应用

构建完成后：

1. **找到安装包**
   - 路径：`bushu1\dist\币安自动交易 Setup 0.1.0.exe`

2. **运行安装程序**
   - 双击安装包
   - 按照安装向导操作：
     - 选择安装目录（可使用默认）
     - 勾选"创建桌面快捷方式"
     - 勾选"添加到开始菜单"
     - 点击"安装"

3. **启动应用**
   - 在桌面找到"币安自动交易"图标，双击打开
   - 或从开始菜单搜索并启动

## 使用指南

### 1. 配置币安 API

1. 进入应用界面，点击"设置"或"API 配置"
2. 输入币安 API Key 和 Secret
3. 选择要监控的合约（如 BTCUSDT、ETHUSDT 等）
4. 点击保存（会自动保存到本地）

**获取 API Key：**
- 登录 [币安官网](https://www.binance.com)
- 进入 "API 管理"
- 创建新的 API Key
- 只需要开通期货交易权限

### 2. 选择交易策略

1. 在应用界面选择策略（当前支持：15分钟趋势+5分钟回调）
2. 配置策略参数（如 EMA 周期、止损止盈比例等）
3. 点击保存

### 3. 启动自动交易

1. 点击"启动监控"按钮
2. 系统会实时扫描市场信号
3. 发现交易信号时自动执行交易

### 4. 监控和调整

- 实时查看持仓、盈亏、交易记录
- 调整策略参数（会立即生效）
- 随时停止自动交易

## 常见问题

### Q1: 脚本运行时提示 "pnpm 不是内部或外部命令"

**A:** 需要先安装 pnpm：
```cmd
npm install -g pnpm
```

### Q2: 构建过程中断或报错

**A:** 
1. 检查网络连接是否正常
2. 重新运行脚本
3. 如果还是失败，请查看错误信息并联系技术支持

### Q3: 应用启动后显示空白页面

**A:** 
1. 检查网络连接是否正常
2. 尝试关闭应用后重新启动
3. 查看应用日志（如有）

### Q4: API 配置后无法连接

**A:** 确认：
- API Key 和 Secret 是否正确
- 是否已开通期货交易权限
- IP 白名单是否设置（如需要）
- 网络连接是否正常

### Q5: 如何卸载

**A:** 
1. 打开"控制面板" → "程序和功能"
2. 找到"币安自动交易" → 点击"卸载"
3. 或在安装目录找到 `unins000.exe` 双击运行

## 手动安装步骤

如果你不想使用自动化脚本，也可以手动执行：

```cmd
# 1. 克隆项目
cd C:\Users\Administrator\Desktop
git clone https://github.com/2nhfymdk77-debug/bushu1.git

# 2. 进入项目目录
cd bushu1

# 3. 安装依赖
pnpm install

# 4. 构建桌面应用
pnpm electron:build

# 5. 安装应用
# 进入 dist 目录，双击运行安装程序
```
